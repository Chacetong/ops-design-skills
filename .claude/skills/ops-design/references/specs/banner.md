# Banner 规格要求

## 尺寸

| 属性 | 值 |
|------|-----|
| 目标尺寸 | 1920 x 960 px |
| 生成尺寸 | 1536 x 1024 px（API 最大，合成阶段裁切/缩放至目标） |
| 宽高比 | 2:1 |
| 输出格式 | PNG |

## 输出方式

**单图直出**：一次性生成包含文字和视觉的完整 Banner，不分层。文字与主视觉融为一体，允许互相穿插和遮挡。

## 构图规范

```
┌────────────────────────────────────────┐
│  ┌─────────┐                           │
│  │ 大标题   │    ★ 主视觉（偏右）        │
│  │ 左上偏   │      装饰元素可散布全图      │
│  │ 占比大   │                           │
│  └─────────┘                           │
│         ↕ 文字与视觉互相穿插             │
└────────────────────────────────────────┘
```

### 构图要点

- **文字位置**：左侧偏上，占画面高度约 50-60%，是画面主角之一
- **主视觉位置**：偏右居中，但不限制在右 55% 内
- **文字与视觉的关系**：互相穿插渗透，主视觉元素可局部遮挡文字，文字也可叠在视觉元素上
- **没有硬分界线**：左右不是"文字区 vs 图片区"，而是统一的全幅场景
- **背景**：全幅统一的场景/色调，不存在"左边留白右边有图"
- **装饰元素**：从主视觉区域向左侧散布，让画面整体丰富

## 安全区域

```
┌─ 60px ─┬──────────────────────┬─ 60px ─┐
│        │                      │        │
├ 40px   │    内容安全区域        │ 40px   │
│        │                      │        │
│        │                      │        │
├ 40px   │                      │ 40px   │
└─ 60px ─┴──────────────────────┴─ 60px ─┘
```
- 上下安全边距：40px
- 左右安全边距：60px
- 所有文字和关键视觉元素不得超出安全区域

## 文字要求

- 标题字号：视觉占比约 50-60% 画面高度，大而有冲击力
- 标题使用英文（AI 绘图模型渲染英文更准确）
- 文字应有强烈的视觉存在感：立体、有材质、有装饰效果
- 文字风格需与画面整体风格统一（如节日主题用暖色立体字，清新主题用白色描边字）

## Prompt 构建指引

单图直出模式下，一个 prompt 需要同时描述画面和文字：

```
[画面场景描述], [主视觉元素及位置],
[装饰元素散布],
large title text "[标题]" in [字体风格] on the left side,
text and visual elements intertwined naturally,
[色彩/氛围], high quality, professional banner, wide 2:1 ratio
```

**关键 prompt 技巧**：
- 明确写出标题文字内容，用引号包裹
- 描述文字的视觉风格（3D、材质、颜色、效果）
- 强调文字与画面的融合关系（intertwined / overlapping / integrated）
- 避免说"text on the left, image on the right"这种硬分区描述
