# 项目风格参考

本文件定义项目整体的视觉调性，适用于所有运营物料（Banner、海报、卡片等）。

> **使用方式**：生成绘图 prompt 时，从本文件提取品牌色、风格调性等关键词融入 prompt。如果用户指定了特定风格偏好，以用户要求为准，本文件作为默认参考。

配色原则：
- 单张物料使用不超过 3 种主要色彩
- 主色占视觉面积 60%，辅色 30%，点缀色 10%
- 促销类物料可用暖色调为主（红、橙、黄）
- 品质/科技类物料以冷色调为主（蓝、灰、白）

## 字体风格

| 场景 | 风格 | Prompt 关键词 |
|------|------|--------------|
| 大标题 | 粗体、现代无衬线 | bold modern sans-serif, clean typography |
| 促销标题 | 圆润、活泼 | rounded playful font, friendly typography |
| 高端/品质 | 细体衬线 | elegant thin serif, luxury typography |
| 手写/节日 | 手写体 | handwritten brush calligraphy |
| 科技/互联网 | 几何无衬线 | geometric sans-serif, tech-style typography |

## 视觉调性

**默认调性**：现代、简洁、专业

- **构图**：留有呼吸感，不拥挤，重要元素有足够的空间
- **光影**：柔和的自然光，避免过于戏剧化的明暗对比
- **纹理**：干净的渐变或微妙的纹理，避免复杂图案
- **图形语言**：圆角矩形、柔和曲线，避免锐利的几何形状

## 摄影 / 插画偏好

- **产品图**：干净背景、柔和阴影、适度的景深
- **人物图**：自然姿态、真实表情，避免过度摆拍
- **插画风格**：扁平化 + 微立体，色彩鲜明但不刺眼
- **背景图**：渐变或抽象几何，避免写实风景照

## 禁忌事项

以下元素/风格应避免：
- 过于复杂的装饰边框和花纹
- 闪烁/发光等廉价感效果
- 文字与背景对比度不足导致难以阅读
- 超过 3 种不同字体的混用
- 过度饱和的荧光色
- 低分辨率的拉伸图片
